\chapter {Design}

Die grundlegenden POA-spezifischen Elemente (Leitungen, E/As, CPUs, Cores und
De-/Multiplexer) werden in der internen Darstellung als Model entworfen,
in der für den Benutzer sichtbaren Darstellung als Views. Als Controller wird
das Qt-spezifische Event-Handling über Signals und Slots gewählt. Hierbei
übernehmen Views eine Signalgeber-Haltung für Benutzerinteraktionen
an die Models, sowie eine Signalempfänger-Haltung zur Darstellung von
Modelveränderungen.\par
